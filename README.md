# REACT-ToDoList 

## 项目运行

### Project setup
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```



## 详细介绍

### 前言

利用react和ant design制作一个ToDoList的demo

### 技术栈

react + redux + react-redux + webpack(create-react-app) + ES6 + ant design + localStorage

### 项目布局

├─src

|  |   |    ├─index.js
|  |   |    ├─showList.js
|  |   |    └toDoList.js
